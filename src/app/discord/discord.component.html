Discord Music Bot

Repo

Discord Music Bot is a bot I started making recently in order to play songs in a server with friends and as a way to learn ES6 and Node.js more.

This bot uses Node.js as the backend and uses the Discord.js library in order to interact with Discord's RESTful API.

This bot can play youtube links by downloading an audio stream and stream it directly through the bot into a Discord channel. It can also queue songs and play entire playlists.

The bot I use for my server is deployed on Heroku.
