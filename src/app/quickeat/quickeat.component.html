QuickEat

Repo

QuickEat is an Android application that I built in two weeks in May 2015 as a final project for CS 135 Introduction to Computer Science 1 and as a fun project to explore Android development.

QuickEat utilizes Google's Geolocation API to find the user's location and queries Google's Places API for restaurants within a 1 mile radius. It then retrieves the list of objects in JSON and QuickEat lists them in the app.

Users are able to call, get directions, and perform other actions through selecting a restaurant in the list.
